# 단지 실거래가 바로보기 — 설계 문서

- 날짜: 2026-07-13
- 상태: 사용자 승인됨
- 관련 파일: `web/components/properties/PriceComparisonSection.tsx`

## 배경 / 문제

매물 상세 페이지의 "실거래가 · 공시지가 공공 데이터" 영역은 외부 사이트(국토부 실거래가 공개시스템, 부동산공시가격 알리미, 네이버 부동산) 링크만 제공한다. 사용자는 외부 사이트로 이동해 지역·단지를 다시 검색해야 한다. 버튼 한 번으로 해당 매물 단지의 실거래 내역을 앱 안에서 바로 볼 수 있어야 한다.

## 범위

- **포함**: 아파트 매매 실거래가 (국토교통부 공공데이터포털 API), 최근 6개월, 버튼 클릭 시 인라인 펼침 표시
- **제외 (YAGNI)**: 공시지가/공동주택 공시가격 API 연동(기존 외부 링크 유지), 전월세 실거래가, 가격 추이 차트, 파이프라인 사전 수집

## 전제 조건

공공데이터포털(data.go.kr)에서 "국토교통부_아파트 매매 실거래가 자료" API 활용신청 후 인증키를 발급받아 `.env.local` 및 Vercel 환경변수에 `MOLIT_API_KEY`로 등록한다. 무료·즉시 승인. 키가 없으면 버튼을 노출하지 않고 기존 외부 링크만 유지한다.

## 데이터 흐름

1. 매물 상세 페이지 공공 데이터 영역에 "⚡ 이 단지 실거래가 바로보기" 버튼 표시 (complex 정보가 있는 매물만).
2. 클릭 시 `GET /api/properties/[id]/transactions` 호출.
3. 서버가 매물의 시군구명(`complexes.sigungu`)을 정적 매핑 테이블(`lib/lawdCodes.ts`)로 법정동 시군구코드(LAWD_CD, 5자리)로 변환.
4. 최근 6개월을 월 단위(DEAL_YMD=YYYYMM)로 조회하되 Supabase 캐시를 우선 확인:
   - `molit_deals_cache` 테이블, 키 `(lawd_cd, deal_ym)`, 해당 시군구·월의 전체 거래를 JSONB로 저장
   - `fetched_at` 기준 24시간 TTL; 같은 시군구 매물끼리 캐시 공유 → 일 1,000건 API 한도 대응
5. 서버가 단지명 매칭(공백·특수문자 제거 정규화 후 부분일치) 필터 + 전용면적 근접순 정렬 후 반환:
   - 거래 리스트: 거래년월, 층, 전용면적(m²), 거래금액(만원)
   - 요약: 거래 건수, 평균가, 현재 매물 가격 대비 차이(%)
6. 프론트엔드는 인라인으로 펼쳐 표시. 0건이면 "최근 6개월 거래 없음" 안내 + 외부 링크 유지.

## 구현 범위

| 파일 | 작업 |
|---|---|
| `web/lib/molit.ts` (신규) | 국토부 API 호출, XML 파싱, 단지명 정규화·매칭, 요약 계산 |
| `web/lib/lawdCodes.ts` (신규) | 시군구명 → 법정동코드 5자리 정적 매핑 |
| `web/app/api/properties/[id]/transactions/route.ts` (신규) | 조회 라우트, 캐시 판단, 응답 조립 |
| `web/components/properties/TransactionHistorySection.tsx` (신규) | 버튼 + 인라인 펼침 UI |
| `web/components/properties/PriceComparisonSection.tsx` (수정) | 공공 데이터 영역에 신규 섹션 삽입 |
| Supabase 마이그레이션 (신규) | `molit_deals_cache` 테이블 — 수동 적용 (운영 방침: SQL 제공 후 직접 실행) |

## UI (인라인 펼침)

```
┌─ 실거래가 · 공시지가 공공 데이터 ───┐
│ [⚡ 이 단지 실거래가 바로보기]       │
│  ▼ 클릭 시 펼침                     │
│  최근 6개월 · 12건 · 평균 8.2억      │
│  2026.06 · 12층 · 84m² · 8억4,000만 │
│  2026.05 · 7층 · 84m² · 8억2,000만  │
│  국토부 실거래가 공개시스템 제공      │
└────────────────────────────────────┘
```

- 현재 매물 가격 대비 차이는 배지(▲/▼ %)로 표시
- 데이터 출처 문구 필수: "국토교통부 실거래가 공개시스템 제공"

## 에러 처리

| 상황 | 동작 |
|---|---|
| `MOLIT_API_KEY` 미설정 | 버튼 미노출, 외부 링크만 표시 |
| 국토부 API 오류/타임아웃 | "잠시 후 다시 시도해 주세요" 메시지 + 외부 링크 안내 |
| 시군구코드 매핑 실패 | 버튼 대신 외부 링크 유지 |
| 단지명 매칭 0건 | "최근 6개월 거래 없음" 안내 |

## 테스트

기존 `web/__tests__` 패턴(vitest)에 맞춰 단위 테스트 추가:

- 단지명 정규화·매칭 (공백/괄호/차수 표기 차이)
- 시군구명 → 법정동코드 변환 (정상/실패 케이스)
- XML 응답 파싱 (정상/빈 결과/에러 응답)
- 요약 계산 (평균가, 현재가 대비 %)
